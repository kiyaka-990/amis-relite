# 🏗️ Amis Relite Limited — Next.js v2

> **Red & Black Glassmorphic Theme · No API Key · Fully Interactive**

## Quick Start
```bash
npm install
npm run dev
# → http://localhost:3000
```
No `.env` needed. Zero API dependencies.

## Features
| | |
|-|-|
| **Theme**     | Deep red & black gradients + heavy glassmorphism throughout |
| **Navbar**    | Floating pill-shaped glass nav, auto-shrinks on scroll |
| **Hero**      | Animated floating particle canvas + glass content card, 4-slide carousel |
| **Stats**     | Animated-in glass pill statistics |
| **About**     | Glassmorphic floating badge & metric pills |
| **Services**  | 6 glass cards with glow ring on hover, icon badges, rounded pill buttons |
| **Projects**  | 3D tilt-on-hover cards with shine effect, color dots, masonry grid, filter tabs |
| **Testimonials** | Auto-carousel with dot navigation, glassmorphic rounded cards |
| **Team**      | Hover bio-reveal cards with animated arrow |
| **FAQ**       | Smooth glassmorphic accordion with numbered questions |
| **CTA**       | Centred glass card over full-bleed image |
| **Contact**   | Glass info cards + rounded glassmorphic form |
| **ARIA Chatbot** | Shake-to-attract FAB · unread badge · date dividers · typing animation · follow-up chips · quick topics · clear button · no API |

## AI Chatbot (ARIA)
Zero external dependencies. Uses a **17-intent pattern-matching engine** in `data/content.ts`.

Topics: services, each service in detail, all 9 projects, pricing, process, timelines, safety, team members, about the company, location/coverage, contact details.

## Deploy
```bash
npm i -g vercel && vercel
```
No environment variables needed.
